Author Box Plugin is Simple and Useful Plugin which anyone can need it. Using this Plugin you can show the author information in a box under the post or in a category.


== Description ==

Author Box Plugin is ability to add the author box in top/bottom of each post or in each category.

About this Plugin:

*   Author Box Plugin is a Simple Plugin and which give an full flexibility to the WordPress Site
*   It contains reasonable customization option in admin settings
*   Easy to use
*   Tested upto WordPress 3.8.2
*   We are always happy to fix the bug in this plugin and enhance the features to this plugin.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `master-mind-author-box/authorboxplugin.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
